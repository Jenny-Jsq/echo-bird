# Echo Bird 浏览器扩展（v2.0.x）

## 简介

Echo Bird（赛博八哥）是一款独特的浏览器插件，它模拟一只从零开始学习说话的小鸟。通过监听你在网页上的日常输入，逐步积累语料库，并以诗意的方式回复你的输入。插件设计为“暗黑陪伴”模式，不打扰你的工作流，而是像一只安静的小鸟，在角落静静观察和回应。

## 设计理念

### 赛博八哥的诞生
- **学习成长**：小鸟从一无所知开始，通过你的真实输入逐步“听到”和“掌握”词汇，形成专属于你的语料库。回复从简单的“啾啾”开始，逐渐融入你常用的词语。
- **句末触发**：采用 5 秒防抖机制，只有当你停止输入后才弹出气泡，避免打断你的思路。中文输入法兼容，确保在候选词选择完成后才开始计时。
- **诗意拼贴**：回复基于已掌握的词汇拼贴而成，风格可选哲理、搞笑或抽象，总字数不超过 10 个汉字，用空格分隔，无标点符号。
- **隐私优先**：本地存储语料库，不上传用户数据；遛鸟模式仅发送当次输入片段用于词条提取。

### v2.0 迭代亮点
- **从零学习**：取消预置语料，小鸟真正从你的输入中成长。
- **智能触发**：句末静默触发 + 输入法兼容，体验更自然。
- **大模型增强**：遛鸟模式下使用 AI 抽取更精准的中文词条。

## 主要功能

- **自动输入监听**：在网页文本框中输入内容，5 秒后自动解析并回复。
- **语料库学习**：词条首次出现为“学习中”，第二次起为“已掌握”，参与回复生成。
- **风格切换**：哲理（偏长词）、搞笑（偏短词）、抽象（随机重组）。
- **遛鸟模式**：集成 OpenAI API，提取更高质量词条；失败时降级到本地分词。
- **设置管理**：在选项页调整风格、开关遛鸟、配置 API Key、清空语料库。
- **角落小鸟**：常驻右下角图标，气泡自动弹出并消失。

## 本地加载（开发）

1. 打开 Chrome（或 Edge）→ `chrome://extensions`
2. 开启「开发者模式」
3. 「加载已解压的扩展程序」→ 选择本目录 `extension/`（包含 `manifest.json` 的文件夹）

### 更新代码后请你本地执行

1. 打开 `chrome://extensions`，找到 Echo Bird，点击 **重新加载**。
2. 若希望语料与「预填占位」彻底脱钩，可在选项页 **清空全部语料** 后再试（或仅重载后在新页面输入框里先 **点一下输入框** 再输入，以便记录「聚焦时」的 baseline）。

## 使用说明（v2.0）

- 在普通文本输入框、多行文本框中输入；**停止输入约 5 秒**后才会解析并弹出气泡（每个输入框 **独立计时**；中文在 **一次合成结束** 后再开始 5 秒倒计时）。
- **聚焦 baseline**：进入输入框时会把当前内容记下来；之后本地分词优先只处理相对 baseline **新增** 的文字，减少把页面预填/占位整段学进语料的情况（中途整段改写时仍按全文处理）。
- **未开遛鸟**时，本地分词 **只保留纯中文词条**（不再把英文单词记入语料）。
- 语料从空开始：词条出现第 1 次仅「听到」，第 2 次起为「已掌握」并可能出现在回复里。回复为 **啾啾**（1～3 个啾）加上已掌握词，总字数不超过约 10（不计空格）。
- **遛鸟模式**：填写 API Key 后，用**大模型**从当次输入中提取中文词条；失败或超时则静默退回本地规则分词。不再用语库诗句生成整句回复。
- 选项中可切换哲理 / 搞笑 / 抽象（影响从已掌握词中的抽样权重）、遛鸟开关、API 与小鸟位置。

## 隐私与安全

- 发送至 API 的内容仅为**当次输入片段**（用于抽词），不含本地语料库全文。
- 扩展会跳过密码框、疑似 URL、邮箱/电话/身份证模式及内置敏感词命中的内容。
